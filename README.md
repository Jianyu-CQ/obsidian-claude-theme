# Obsidian Claude Theme 🟠

A minimal, warm-toned Obsidian CSS snippet inspired by Claude's web interface.

## Features
- Warm beige background (#F5F0E8)
- Claude orange accent color (#C96442)
- Centered 720px reading layout
- Styled code blocks, blockquotes, tables, callouts

## Installation
1. Download `claude-theme.css`
2. Place it in your vault's `.obsidian/snippets/` folder
3. Go to Settings → Appearance → CSS Snippets → Enable `claude-theme`
